# Zero-Baseline Vendor Performance Demo

## How to Test Zero-Baseline Performance

1. **Go to Vendors page**
2. **Click "Add Vendor"**
3. **Add a new vendor** (e.g., "NEW VENDOR TEST")
4. **Observe the results:**

### Expected Zero-Baseline Behavior:
- **Each KPI Score**: 0/3 (gray color)
- **Total Score**: 0/15
- **Overall Rating**: "Not Rated" (gray badge)
- **Work Order Stats**: All zeros

### Comparison with Existing Vendors:
- **POWER**: Has work orders → Shows actual KPI scores (1-3)
- **AIR PURIFIERS**: Has work orders → Shows actual performance
- **NEW VENDOR**: No work orders → Shows 0/3 for all KPIs

### Side-by-Side Comparison:
1. **Select 2 vendors** (one with work orders, one without)
2. **Compare performance** side-by-side
3. **Zero-baseline vendor** shows:
   - All KPI bars at 0%
   - 0 work orders in all categories
   - No SLA performance data

This demonstrates that **vendors earn performance only through actual work**, not through default ratings.
